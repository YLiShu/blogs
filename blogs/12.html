    深拷贝与浅拷贝

    黎子 2023/11/13 Javascript

    # 深拷贝和浅拷贝

    在 JavaScript 中，对象和数组是引用类型，即它们在内存中的地址是一样的。当我们对一个引用类型的变量进行赋值或传递时，实际上是将该变量的地址传递给另一个变量。这就导致了一个问题：如果我们对一个对象或者数组进行修改，会同时影响到所有引用该对象或数组的变量。为了避免这个问题，我们需要使用拷贝（或者复制）来创建新的对象或数组。在 JavaScript 中，拷贝分为深拷贝和浅拷贝两种方式。

    # 浅拷贝

    浅拷贝是指仅仅拷贝对象或数组的引用，而不是拷贝对象或数组本身。也就是说，新对象和原对象共享一个内存地址，对新对象的修改会影响到原对象。

    在 JavaScript 中，我们可以使用以下方式进行浅拷贝：

    # Object.assign()

    Object.assign() 方法用于将一个或多个源对象的所有可枚举属性复制到目标对象中。它返回目标对象。

    const originalArray = [
	{ name: 'John', age: 30 },
	{ name: 'Alice', age: 25 },
];

const newArray = Object.assign([], originalArray);
newArray[0].age = 35;
console.log(originalArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

console.log(newArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

    在这个例子中，我们使用 Object.assign() 方法将 originalArray 的所有属性复制到一个新对象中，从而实现了浅拷贝。

    # 扩展运算符

    扩展运算符 ... 用于将一个数组或对象展开成一组逗号分隔的值。在浅拷贝中，我们可以使用扩展运算符来拷贝一个数组或对象。

    const originalArray = [
	{ name: 'John', age: 30 },
	{ name: 'Alice', age: 25 },
];

const newArray = [...originalArray];
newArray[0].age = 35;

console.log(originalArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

console.log(newArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

    在这个例子中，我们使用扩展运算符将 originalArray 数组展开成一组逗号分隔的值，从而实现了浅拷贝。

    # 数组的 slice() 和 concat() 方法

    数组的 slice() 方法用于返回一个新数组，其中包含从开始位置到结束位置之间的元素。如果省略结束位置，则返回从开始位置到数组末尾的元素。数组的 concat() 方法用于将多个数组合并成一个数组。

    const originalArray = [
	{ name: 'John', age: 30 },
	{ name: 'Alice', age: 25 },
];

const newArray = originalArray.slice();
newArray[0].age = 35;

console.log(originalArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

console.log(newArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

const originalArray = [
	{ name: 'John', age: 30 },
	{ name: 'Alice', age: 25 },
];

const newArray = [].concat(originalArray);
newArray[0].age = 35;

console.log(originalArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

console.log(newArray);
// 输出: [ { name: 'John', age: 35 }, { name: 'Alice', age: 25 } ]

    在这个例子中，我们使用数组的 slice() 方法和 concat() 方法来实现浅拷贝。

    # 深拷贝

    深拷贝是指完全拷贝对象或数组本身，而不是仅仅拷贝对象或数组的引用。也就是说，新对象和原对象完全独立，互不影响。

    在 JavaScript 中，我们可以使用以下方式进行深拷贝：

    # JSON.parse() 和 JSON.stringify()

    JSON.parse() 方法用于将 JSON 字符串解析为 JavaScript 对象。JSON.stringify() 方法用于将 JavaScript 对象转换为 JSON 字符串。通过将对象转换为 JSON 字符串，我们可以实现深拷贝。

    # JSON.parse()

    JSON.parse() 方法用于将一个 JSON 字符串转换为一个 JavaScript 对象。当使用 JSON.parse() 解析一个 JSON 字符串时，它会递归地遍历整个 JSON 结构，并为每个对象创建一个新的 JavaScript 对象，包括所有嵌套的对象和数组。 在解析过程中，JSON.parse() 会为每个对象创建一个新的引用，并将 JSON 中的值复制到新的 JavaScript 对象中。这样，原始的 JSON 字符串和最终生成的 JavaScript 对象是完全独立的。

    由于这个递归过程，JSON.parse() 可以处理嵌套的对象和数组，确保所有内部对象也都是深度复制的。因此，JSON.parse() 是深拷贝。

    # JSON.stringify()

    JSON.stringify() 方法用于将一个 JavaScript 对象转换为一个 JSON 字符串。当使用 JSON.stringify() 将一个 JavaScript 对象转换为 JSON 字符串时，它也很递归地遍历整个对象结构，并将每个对象的值转换为 JSON 格式。 在转换过程中，JSON.stringify() 会复制对象的值，而不是引用。如果对象的值是基本类型（如字符串、数字、布尔值的等）或者 null，它们会被直接复制到生成的 JSON 字符串中。对于对象和数组，JSON.stringify() 会递归调用，确保它们内部的 对象和数组也是深度复制的。 由于这个递归过程，JSON.stringify() 可以处理嵌套的对象和数组，并且确保所有内部对象都是深度复制的。因此，JSON.stringify() 也是深拷贝。

    const obj1 = { a: 1, b: { c: 2 } };
const obj2 = JSON.parse(JSON.stringify(obj1));
console.log(obj2); // 输出 { a: 1, b: { c: 2 } }

    在这个例子中，我们先使用 JSON.stringify() 方法将 obj1 对象转换为 JSON 字符串，然后使用 JSON.parse() 方法将 JSON 字符串解析为新的对象 obj2，从而实现深拷贝。需要注意的是，如果对象中包含函数、正则表达式等特殊类型的属性，这种方法可能会出现问题。

    需要注意的是，JSON.parse() 和 JSON.stringify() 在处理循环引用时会抛出异常。循环引用是指对象或数组之间形成了循环，导致无限递归。因此，在使用 JSON.parse() 和 JSON.stringify() 进行深拷贝时，要确保数据中没有循环引用。对于包含循环引用的数据结构，需要使用其他方法或者库来实现深拷贝。

    # 递归拷贝

    递归拷贝是指对于对象或数组中的每一个子元素，都进行一次拷贝操作。通过递归拷贝，我们可以实现深拷贝。

    function deepClone(obj) {
	if (typeof obj !== 'object' || obj === null) {
		return obj;
	}

	const result = Array.isArray(obj) ? [] : {};

	for(const key in obj) {
		if (Object.hasOwnProperty.call(obj, key)) {
			result[key] = deepClone(obj[key]);
		}
	}

	return result;
}

const obj1 = {
	a: 1,
	b: {
		c: 2
	}
};
const obj2 = deepClone(obj1);
console.log(obj2); // 输出 { a: 1, b: { c: 2 } }

    在这个例子中，我们定义了一个 deepClone() 函数来实现递归拷贝。对于传入的对象或数组，如果它是基本类型或 null，则直接返回。否则，我们创建一个新的对象或数组，并将原对象或数组中的每一个子元素都进行一次拷贝操作，最终返回新的对象或数组。

    需要注意的是，递归拷贝可能会导致栈溢出或循环引用等问题。因此，在实际应用中，需要根据具体情况选择合适的拷贝方式。
