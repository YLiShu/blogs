    toPrecision和toFixed以及Math.round的区别

    黎子 2023/11/13 Javascript

    # toPrecision 和 toFixed 和 Math.round 有什么区别？

    # toPrecision()

    • toPrecision() 方法用于将数字转换为指定长度的字符串表示。
    • 参数是一个整数，表示返回结果的总位数（包括整数部分和小数部分）。
    • 结果会进行四舍五入。
    • 示例：
    let num = 123.456789;
num.toPrecision(5); // "123.46"

    # toFixed()

    • toFixed() 方法用于将数字转换为指定小数位数的字符串表示。
    • 参数是一个整数，表示返回结果的小数位数。
    • 结果会进行四舍五入。
    • 示例：
    let num = 123.456789;
num.toFixed(2); // "123.46"

    # Math.round()

    • Math.round() 方法用于将数字四舍五入为最接近的整数。
    • 接受一个数字作为参数，并返回最接近的整数。
    • 示例：
    Math.round(123.456789); // 123

    # 总结：

    • toPrecision() 和 toFixed() 方法都可以将数字转换为字符串，但 toPrecision() 方法可以指定总位数，而 toFixed() 方法只能指定小数位数。
    • toPrecision() 和 toFixed() 方法都会四舍五入
    • Math.round() 方法只能将数字四舍五入为最接近的整数，不能指定位数。
