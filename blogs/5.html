    # 牛客模拟·大厂定制真题前端

    假设文本文件中有多处连续的空白行，下边哪个命令可以压缩相邻的空白行？ A. cat -b file B. cat -n file C. cat -s file D. cat -T file

    答案：cat -s file 解析：

    1. cat -s 可以压缩多余空白行
    2. cat -T 是将制表符显示为 ^|
    3. cat -n 显示行号
    4. cat -b 显示行号，但是空白行不显示行号

    如下 js 代码会输出什么？

    console.log(1);
async function fun1(){
	console.log(6);
	await fun2();
	console.log(7);
}

setTimeout(()=>{
	console.log(9);
},0);

async function fun2(){
	new Promise((resolve)=>{
		console.log(2);
		resolve(3);
		setTimeout(()=>{
			console.log(4);
		},0);
	}).then((data)=>{
		console.log(data)
	}).then(()=>{
		console.log(5);
	});
}

fun1()
console.log(8);

    答案：1 6 2 8 3 7 5 9 4 解析：

    1. 根据 js 的执行机制，首先执行同步代码，即打印 1
    2. 继续往下执行，将 setTimeout 放入宏队列等待执行
    3. 继续往下执行，调用 fun1() 函数，打印 6
    4. 执行 await fun2() 函数，输出 2 后，await 会让出当前线程，将后面的代码加到任务队列中，继续执行 fun1() 函数后面的同步代码，即输出 8
    5. 再继续往后执行 fun2()，将第二个 setTimeout 放入宏队列末位等待执行，执行第一个 then 打印 3
    6. 再跳出 fun2() 函数，执行 await fun1() 后面的打印 7
    7. 往后执行 fun2() 的第二个 then，输出 5，至此，微任务全部执行完，开始执行宏任务
    8. 依次输出 9 和 4

    分别以下列序列构造二叉排序树，层序遍历与其他三个不同的是？ A. (100，80，90，140，50，120，160) B. (100，140，160，120，80，50，90) C. (100，80，50，90，140，160，120) D. (100，50，90，80，120，140，160)

    答案：D. (100, 50, 90, 80, 120, 140, 160) 解析：

    1. 二叉排序树的特点，左子树的所有节点都比根节点小，右子树的所有节点都比根节点大，左、右子树也是个二叉排序树

    以下代码执行的结果是？

    function Animal() {}
Animal.price = 1000;
Animal.prototype.price = 20;
var Dog = function () {};
Dog.prototype = new Animal();
var cat = Animal;
var dog = new Dog();

console.dir(dog.price);
console.log(cat.price);

    答案：20，1000 解析：dog 的构造函数是 Dog，Dog 的 prototype 是 new Animal()，即执行 new 操作符会产生对象，将 prototype 对象挂在 __proto__ 下，即 prototype.price 被继承，所以 dog.price 是存在的且为 20，而 animal.price = 1000 被放在了 constructor 里，当在浏览器里打印 dog 时，会看到这样的结构：

    而 cat 只是个新定义的函数，即 Animal，拥有一个 price 属性，且值为 1000 简言之：dog 对象上找不到的属性需要到 dog 的原型链上找，cat 实际上就是 Animal，cat.price === Animal.price，这是 Animal 的静态属性

    对于选择重传协议（SR），若发送窗口最大尺寸为16，问序号位数为？

    答案：5 解析：$W_{Tmax} = W_{Rmax} = 2^{(n-1)}$，其中 n 表示序号位数，Tmax 表示发送窗口的最大尺寸，而 Rmax 表示接收窗口的最大尺寸。
