    判定字符是否唯一

    # 判定字符是否唯一

    实现一个算法，确定一个字符串 s 的所有字符是否全都不同。

    示例 1：

    输入: s = "leetcode" 输出: false

    示例 2：

    输入: s = "abc" 输出: true

    限制：

    • 0 <= len(s) <= 100
    • s[i]仅包含小写字母
    • 如果你不使用额外的数据结构，会很加分

    使用哈希表

    该方法使用一个 HashSet 来存储已经出现过的字符，如果在遍历过程中发现重复的字符，则返回 false。这种方法的时间复杂度为 O(n)，其中 n 是字符串的长度。

    class Solution {
    public boolean isUnique(String astr) {
        HashSet<Character> set = new HashSet<>();
        for (char c : astr.toCharArray()) {
            if (set.contains(c)) {
                return false;
            }
            set.add(c);
        }
        return true;
    }
}

    暴力

    该方法使用两个嵌套循环来比较每个字符与后面的字符是否相同，如果找到相同的字符，则返回 false。这种方法的时间复杂度为 O(n^2)，其中 n 是字符串的长度。

    class Solution {
    public boolean isUnique(String astr) {
        for(int i = 0; i < astr.length(); ++i) {
            for (int j = i + 1; j < astr.length(); ++j) {
                if (astr.charAt(i) == astr.charAt(j)) {
                    return false;
                }
            }
        }
        return true;
    }
}

    位运算

    在这个算法中，我们使用一个 32 位的整数 checker 来表示字符的出现情况。每个字符对应一个位，如果该字符出现过，则对应位上的值为 1，否则为 0。

    我们遍历字符串中的每个字符，将其转换为对应的位索引（通过减去字符 a 的 ASCII 得到相对索引）。然后，我们使用位运算检查对应位上的值是否已经为 1，如果是，则表示该字符已经出现过，返回 false；否则，将对应位上的值设置为 1。

    这种方法的时间复杂度为 O(n)，其中 n 是字符串的长度。它不使用额外的数据结构，并且适用于只包含小写字母的字符串。

    class Solution {
    public boolean isUnique(String astr) {
        int checker = 0;
        for (int i = 0; i < astr.length(); ++i) {
            int val = astr.charAt(i) - 'a';
            if ((checker & (1 << val)) > 0) {
                return false;
            }
            checker |= (1 << val);
        }
        return true;
    }
}

    上述代码中，int val = astr.charAt(i) - 'a'; 通过将字符减去小写字母 'a' 的值来得到对应的整数值。例如，字符 'a' 被转换为 0，字符 'b' 被转换为 1，以此类推。然后使用 1 << val，使得 val 只有对应位为 1，再使用 (checker & (1 << val)) > 0 检查是否已经出现过。例如，当 val 为 1 时，1 << val 的结果为 0010，checker 为 0000，checker & (1 << val) 为 0000（值为 0），表示未出现过，然后使用 checker |= (1 << val) 使得 checker 对应的 val 位为 1。

    常见的位运算操作符：

    1. 与运算（&）：对两个操作数的每个对应位执行逻辑与操作，如果两个位都是 1，则结果为 1，否则为 0。 例如：5 & 3 的二进制表示为 0101 & 0011，结果为 0001，即1。
    2. 或运算（|）：对两个操作数的每个对应位执行逻辑或操作，如果两个位中至少有一个为 1，则结果为 1，否则为 0。 例如：5 | 3 的二进制表示为 0101 | 0011，结果为 0111，即7。
    3. 异或运算（^）：对两个操作数的每个对应位执行逻辑异或操作，如果两个位不相同，则结果为 1，否则为 0。 例如：5 ^ 3 的二进制表示为 0101 ^ 0011，结果为 0110，即 6。
    4. 取反运算（~）：对操作数的每个位执行逻辑非操作，将 1 变为 0，将 0 变为 1。 例如：~5 的二进制表示为 ~0101，结果为 1010，即 -6（以补码形式表示）
    5. 左移运算（<<）：将操作数的二进制表示向左移动指定的位数，右侧用 0 填充。 例如：5 << 2 的二进制表示为 0101 << 2，结果为 010100，即 20。可以看做乘以 2 的多少次幂。但需要注意，左移运算的结果可能会溢出或丢失精度，因此在使用左移运算时，应该根据具体的需求和数据类型进行适当的处理和验证。
    6. 右移运算（>>）：将操作数的二进制表示向右移动指定的位数，左侧用符号位填充（对于正数用 0 填充，负数使用 1 填充）。 例如：10 >> 2 的二进制表示为 1010 >> 2，结果为 0010，即 2。
